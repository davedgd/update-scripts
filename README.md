# update-scripts
Update scripts for various OSes

# Windows: update.bat
- Chocolatey
- Windows Store
- Python (via conda/pip)
- R (binaries)
- Windows Updates

# Mac: update.sh
- TeX
- Brew (including Casks)
- Python (via conda/pip)
- R (binaries)
- macOS Updates

# Ubuntu Linux: update-ubuntu.sh
- Ubuntu Updates (via apt)
- Python (via pip-review)
- R (sources)
